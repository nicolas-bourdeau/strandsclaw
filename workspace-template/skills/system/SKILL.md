# System Operations

Use this skill when coordinating StrandsClaw runtime behavior, bootstrapping workspace context, and deciding where new code belongs.

- Keep runtime code lean and tied to active features.
- Prefer workspace-first behavior over repository-local assumptions.
- Use spec artifacts for non-trivial implementation changes.
